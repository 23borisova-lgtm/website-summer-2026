# Сайт летнего лагеря ТриКита «Энциклопедия Невозможного»

Лендинг летнего городского лагеря (Пенза). Деплой — Vercel: https://website-summer-2026.vercel.app/

## Как устроено

- **`index.src.html`** — ИСХОДНИК, который редактируем (React + JSX, как раньше). Меняем тексты, цены, смены — здесь.
- **`build.mjs`** — сборка: компилирует JSX заранее и предрендерит страницу в готовый HTML (чтобы Яндекс и Google видели весь текст и сайт грузился быстрее).
- **`index.html`** и **`app.js`** — РЕЗУЛЬТАТ сборки. Не редактируем вручную — они перезаписываются при сборке. Именно их отдаёт Vercel.

## Как внести изменения

1. Правим `index.src.html`.
2. Один раз ставим зависимости: `npm install`
3. Собираем: `npm run prerender` (генерирует `index.html` и `app.js`)
4. Коммитим и пушим — Vercel отдаёт уже готовые файлы (сборку на своей стороне НЕ запускает, см. `vercel.json`).

## Что нужно дозаполнить

- **Яндекс.Метрика** — счётчик 109919770 уже подключён (Вебвизор, цели: `lead`, `booking_click`, `telegram`, `phone`).
- **Форма заявки** → функция `api/lead.js`: создаёт лид в **AlfaCRM** и шлёт уведомление админу в **MAX**.
  Чтобы заработало, в Vercel (Settings → Environment Variables) нужно задать переменные из `.env.example`:
  `ALFACRM_HOST`, `ALFACRM_EMAIL`, `ALFACRM_API_KEY`, `ALFACRM_BRANCH`, и для MAX — `MAX_BOT_TOKEN` + `MAX_ADMIN_USER_ID`.
  После сохранения переменных сделать Redeploy.
